            第一步：下载安装git

            1、官方下载地址：https://git-scm.com/downloads

            2、在服务器和本地电脑上，都要安装Git，安装步骤如下图：

            第二步、在github上创建仓库

            1、登录https://github.com/，没有账号的，请先注册

            2、在页面上方用户菜单上选择 “+” → New repository 创建一个新的仓库

            3、为仓库取一个名字，点击创建仓库按钮，你将成功创建一个仓库，create。 

            第三步、将本地项目文件上传到git仓库

            1、先进入项目文件夹，鼠标右键选择“Git Bash”，输入命令 git init 把这个目录变成git可以管理的仓库

            2、配置用户身份信息（所有者的标签），这样在提交代码的时候，Git就可以知道是谁提交的，命令如下：

            git config --global user.name "xiao7"
git config --global user.email "15577969@qq.com"

            输入下方的命令，可以查看全部的用户信息：

            cat ~/.gitconfig

            3、用命令 git add .添加到暂存区里面去，不要忘记后面的小数点“.”，意为添加文件夹下的所有文件，如下：

            3、用命令 git commit告诉Git：把文件提交到仓库，引号内为提交说明

            git commit -m '第一次提交'

             4、关联到git远程仓库

            git remote add origin 你的远程库地址

            5、输入push推送命令：

            git push -u origin master

             假如是提交到github的远程库，按要求输入github的账号和密码，即可完成上传。

            第四步、从git远程仓库获取项目代码

            打开Git Bash（需要先安装Git客户端），输入下方的命令：

            git clone 你的远程库地址

             假如获取的是github远程库上的项目，需要输入github的账号和密码才可以。

            第五步、提交项目代码到git远程仓库

            1、打开Git Gui（需要先安装Git客户端）,点击“Rescan”按钮，可以扫描出本地和远程仓库上不一样的文件。

            2、点击Unstaged Changes里改变的文件，然后填写提交备注信息，再点击“Commit”按钮，即可完成代码提交

            3、提交完成后，还需要push推送到远程仓库上，才是真正的完成
